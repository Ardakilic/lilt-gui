## [Unreleased]

### Added
- Initial release of Lilt GUI
- Modern cross-platform GUI built with Wails v2
- React + TypeScript frontend with DaisyUI components
- Multi-language support (English, Turkish, German, Spanish)
- Docker mode support for easy transcoding without installing external tools
- Configuration persistence across sessions
- Real-time transcoding output display
- Binary path auto-detection from system PATH
- Browse dialogs for selecting files and directories
- Format enforcement options (FLAC, MP3, ALAC)
- Metadata preservation option
- Image copying option
- Stop transcoding functionality
- Comprehensive test suite with >80% coverage
- Docker-based development workflow via Makefile
- GitHub Actions CI/CD for multi-platform builds
- Support for Windows, macOS (Intel + Apple Silicon), and Linux (AMD64 + ARM64)
